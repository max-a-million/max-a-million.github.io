Effectivety analisys of algorithms


Each algorithm can be charaterized at least two types of efficency criterias: time and space.
Time efficency helps us to estimate time of execution of the algorithm.
Space efficency has deal with additional memory, wich require for the algorithm execution.
Additionaly, we can apply each criteria of efficency for different cases of execution the algorithm:
  • Best efficency
  • Worst efficency
  • Common efficency
  • Amortized efficency
